# Luca Project
This project is used to train and create AI models with custom datasets.

## Requirements
- Python 3.9+
- FastAPI
- Vue.js

## Setup
1. Install dependencies: `pip install -r requirements.txt` and `npm install` in the vue directory
2. Run the FastAPI server: `uvicorn main:app --host 0.0.0.0 --port 8000`
3. Run the Vue.js application: `npm run serve` in the vue directory
